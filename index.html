Домашка
  1. A B-В качестве селектора может выступать любой элемент HTML, для которого определяются правила форматирования, такие как: цвет, фон, размер и др.
  2. #Id-Идентификатор определяет уникальное имя элемента, которое используется для изменения его стиля и обращения к нему через скрипты.
  3. .class-применяют,когда необходимо определить стиль для одного или нескольких элементов веб-страницы.Можно добавлять несколько классов к 1 элементу через пробел
  4. *-устанавливает стиль для всех элементов веб-страницы. Универсальный селектор, которий соотвествует любому эелементу веб-страницы
  5. Вложенные селекторы-E F {Описание правил стиля}. Применяет стиль для вложенного элемента,при соблюдении вложенности кода.
  6. A > B(Дочерние селекторы)Стиль применяется к элементу В, только когда он является дочерним для элемента А.
  7. A + B -Для управления стилем соседних элементов используется символ плюса(+),который устанавливается между двумя селекторами E и F. Пробелы вокруг плюса не обязательны. Стиль при такой записи применяется к элементу F, но только в том случае, если он является соседним для элемента E и следует сразу после него.
  8. A ~ B-похожи на "+",но в отличие от них стилевые правила применяются ко всем близлежащим элементам
  9. [attr]-Стиль применяется к тем элементам, внутри которых добавлен указанный атрибут. Пробел между именем селектора и квадратными скобками не допускается.
  10. [attr^='value']- начало с "указанного" текста. Можно добавлять к определенным селекторам f.e:E[атрибут^=" <значение>"]
  11. [attr$='value']-окончание "указанного" текста
  12. [attr~='value']-Некоторые значения атрибутов могут перечисляться через пробел, например имена классов. Чтобы задать стиль при наличии в списке требуемого значения требуется использовать конструкцию ~=.
